# ShadowMarket - Build progress

This repository contains the ShadowMarket marketplace scaffold. A build helper (build_zip.php) is included to package the project into a ZIP file ready for distribution. Run `php build_zip.php` from repository root to create the ZIP.

Make sure to import all SQL files in order:
- sql/schema.sql
- sql/schema_additions.sql
- sql/schema_more.sql
